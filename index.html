Sistema de puntos

5 ptsMarcador exacto. Ejemplo: predicción 2-1 y resultado real 2-1.
3 ptsGanador correcto sin marcador exacto. Solo aplica si hubo ganador: local o visitante.
0 ptsFallo con partido finalizado o empate no exacto. Los partidos pendientes no suman y no cuentan como fallo.

Empates

  • Real 1-1 y predicción 1-1: 5 puntos.
  • Real 1-1 y predicción 0-0: 0 puntos.
  • Real 0-0 y predicción 2-2: 0 puntos.
  • Predicción de empate contra victoria real: 0 puntos.

Criterios de desempate

  1. Mayor puntaje total.
  2. Más marcadores exactos de 5 puntos.
  3. Más aciertos de ganador de 3 puntos.
  4. Nombre en orden alfabético.
